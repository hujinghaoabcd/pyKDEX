# Installation

```bash
python -m pip install -e ".[test]"
```

Geospatial export requires:

```bash
python -m pip install -e ".[geo]"
```


## Network data support

```bash
python -m pip install -e ".[network]"
```

Online OpenStreetMap acquisition through OSMnx is optional:

```bash
python -m pip install -e ".[osm]"
```
